# Example Data
This folder contains a sample HR-ToF-AMS dataset for testing 
the DFG workflow.

## File Description
- `Example_a_pinene_SOA.csv`: A sample mass spectrum with columns 
  `mass`, `formula`, and `frac_abundance`, normalized to a sum of 1.

## Expected Output
Running the DFG notebooks with this file should produce outputs 
similar to the example figures shown in the main README.
