# AMS_delta_functionalgroups

General information: two code blocks are included here for delta analysis of AMS data. The first generates radial delta plots for carbon, hydrogen, and oxygen containing fragments. The second calculates the functional group distribution for an AMS mass spectrum. 
To use these code blocks, data sets must be .CSV files with columns that are labeled mass, formula, and frac_abundance. The data should be normalized to a sum of one for all the ion intensities.
